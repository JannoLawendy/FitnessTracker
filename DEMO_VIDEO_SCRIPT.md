# 5-Minute Demo Video Script

1. Introduce the project: Trailblazer Fitness Tracker.
2. Explain the goal: helping users track workouts and progress.
3. Show the home screen and dashboard statistics.
4. Add a new workout and explain validation.
5. Go to the history screen and show the saved workout.
6. Edit the workout and show the updated result.
7. Delete a workout and show the confirmation dialog.
8. Open the progress screen and explain total workouts, minutes, calories, and average time.
9. Briefly explain the code structure: MVVM, Room, Repository, ViewModel, Compose screens.
10. End by saying the app meets the final project requirements.
