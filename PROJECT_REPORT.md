# Fitness Tracker App — One Page Report

## Project Overview
The Fitness Tracker App helps users record their workouts and monitor their progress. Users can add workout information such as title, type, duration, calories, and date. The app is simple and easy to use.

## Main Features
- Add workouts
- Edit workouts
- Delete workouts
- View workout history
- See total workouts, total calories, and total duration

## Technologies Used
- Kotlin
- Jetpack Compose
- Room Database
- MVVM Architecture
- Navigation Compose

## Architecture
The project uses MVVM. The UI screens display information, the ViewModel manages app state, and the Room database stores workout data.

## Future Improvements
Future improvements can include workout reminders, progress charts, login system, cloud synchronization, and personalized workout suggestions.
